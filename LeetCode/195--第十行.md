# leetcode 195 验证回文串
## 第十行
```
假设 file.txt 有如下内容：
Line 1
Line 2
Line 3
Line 4
Line 5
Line 6
Line 7
Line 8
Line 9
Line 10
你的脚本应当显示第十行：
Line 10
```
```sh
#sed 可以解决少行的问题
sed --quiet {10p} file.txt
#如果先用head取前10行，再用tail取最后一行，不能解决少行问题
head --lines=10 file.txt | tail --lines=1
#如果先用tail去取第十行以后的尾部，再用head去取第一行可以解决少行的问题
tail --lines=+10 file.txt | head --lines=1
#awk
awk 'NR==10' file.txt
```
