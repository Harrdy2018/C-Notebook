## 腾讯 测试开发面试
* 宏定义求数组长度(答对)
```c
#include<stdio.h>
#define ARRAY_SIZE(ARRAY) (sizeof(ARRAY)/sizeof(ARRAY[0]))
int main(void) {
    int arr[]={1,2,3,4,5};
    printf("%d\n",ARRAY_SIZE(arr));//5
    char str[]="12345";
    printf("%d\n",ARRAY_SIZE(str));//6
    return 0;
}
```
* 脑筋急转弯
```
有三个箱子。一个里面全部是苹果，一个里面全部是橘子，还有一个苹果和橘子都有。
这三个箱子都贴错了标签，没法从标签上知道里面的内容物。
你只可以打开一个箱子，不可以往里头看，拿出一个水果。看着这个拿出来的水果，你如何立即把箱子和正确的标签对应起来？
```
```
不能拿贴苹果或者橘子的，因为拿这两个标签的话，里面可能会有混合的，根本区分不出来。
拿混合标签的箱子，标签为混合，里面的东西都是单一的，
    混合标签装的是苹果，橘子标签装的是(苹果或者混合物，那就是混合物)，苹果标签装的是(橘子)
    混合标签装的是橘子，苹果标签装的是(橘子或者混合物，那就是混合物)，橘子标签装的是(苹果)
```
